# Sales Performance Dashboard - Power BI

## Project Overview
This project is a Power BI Sales Performance Dashboard developed for a regional sales company to support monthly business reviews.

The dashboard helps management understand:
- Overall business performance
- Year-over-Year sales and profit growth
- Regional performance
- Manager performance
- Category-wise target achievement
- Sales vs Target analysis
- Product returns
- Delivery delays
- Business risks and action areas

## Dashboard Pages

### 1. Executive Overview
This page provides a high-level business summary with:
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Return %
- Total Target
- Achievement %
- Sales Growth %
- Profit Growth %
- Sales by Month and Year
- Sales by Region
- Sales by Category
- Profit by Category

### 2. Manager Performance
This page focuses on manager-level performance and includes:
- Manager Sales Performance
- Manager Profit Performance
- Sales vs Target by Manager
- Target Achievement by Manager
- Category-wise manager performance matrix

### 3. Operations & Risk
This page focuses on returns and delivery performance:
- Returned Orders by Category
- Returned Orders by Region
- Average Delivery Days by Ship Mode
- Delivery Delay by Region
- Risk table with Region, Category, Return %, and Avg Delivery Days

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## Business Goal
The objective of this dashboard is to help the Sales Manager:
- Understand whether the business is growing
- Evaluate profitability
- Track target achievement
- Compare manager performance
- Identify return-related issues
- Monitor delivery delays
- Highlight key risk areas for action
